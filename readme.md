# travail HTML

## Date limite
lundi 18 octobre, 23:00

## Événement de cours

- Vous devez toujours utiliser des fichiers de type .php (pour la partie html)

- Créez une site avec un minimum de trois pages :

```
  /Accueil
  /Produits
  /Qui suis-je / qui sommes-nous ?
```
- Il faut au minimum intégrer :

```
  - Une liste
  - Un tableau
  - Des images
  - Des titres
  - Des balises HTML 5
  - Des liens
```
